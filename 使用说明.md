# YouTube Downloader 使用说明

欢迎使用 YouTube Downloader！本工具支持在 macOS 和 Windows 操作系统上下载 YouTube 视频。

由于本应用程序是个人开发，未经过官方数字签名，因此在首次运行时，您的操作系统可能会弹出安全警告。请按照以下说明操作，以确保应用程序能够正常运行。

---

## 1. macOS 系统用户 (针对 .app 文件)

在 macOS 系统上，当您尝试打开一个未签名的应用程序时，系统可能会提示“无法打开‘YouTube Downloader.app’，因为它来自身份不明的开发者。”这是 macOS 的 Gatekeeper 安全机制。

请按照以下步骤绕过此警告：

1.  **找到应用程序**：
    *   在 Finder 中，导航到您下载的 `YouTube Downloader.app` 文件所在的位置（通常在 `dist/` 文件夹内）。

2.  **右键点击并选择“打开”**：
    *   **不要直接双击**。请按住键盘上的 `Control` 键，然后用鼠标点击 `YouTube Downloader.app` 文件（或者直接用两指点击触控板）。
    *   在弹出的菜单中，选择“**打开**”。

3.  **确认打开**：
    *   此时会弹出一个对话框，再次询问您是否确定要打开此应用程序。
    *   点击对话框中的“**打开**”按钮。

4.  **完成**：
    *   应用程序现在应该会正常启动。
    *   此操作只需执行一次。之后，您就可以像打开其他应用程序一样，直接双击 `YouTube Downloader.app` 来启动它了。

---

## 2. Windows 系统用户 (针对 .exe 文件)

在 Windows 系统上，当您尝试运行一个从互联网下载的、未签名的 `.exe` 文件时，Windows Defender SmartScreen 可能会弹出安全警告。

请按照以下步骤绕过此警告：

1.  **找到应用程序**：
    *   在文件资源管理器中，导航到您下载的 `YouTube Downloader.exe` 文件所在的位置（通常在 `dist/` 文件夹内）。

2.  **点击“更多信息”**：
    *   双击 `YouTube Downloader.exe` 文件。
    *   如果弹出“Windows 已保护你的电脑”或“Windows Defender SmartScreen 阻止了无法识别的应用启动”的警告窗口，请点击窗口左下角的“**更多信息**”（或“More info”）链接。

3.  **点击“仍要运行”**：
    *   点击“更多信息”后，您会看到一个“仍要运行”（或“Run anyway”）的按钮。
    *   点击此按钮。

4.  **完成**：
    *   应用程序现在应该会正常启动。
    *   此操作只需执行一次。之后，您就可以像打开其他应用程序一样，直接双击 `YouTube Downloader.exe` 来启动它了。

---

## 3. 常见问题与故障排除

*   **应用程序无法启动**：
    *   请确保您已按照上述步骤正确绕过了安全警告。
    *   检查您的杀毒软件或防火墙设置，有时它们可能会误报并阻止应用程序运行。您可以尝试暂时禁用它们，然后再次运行应用程序。
    *   如果问题依然存在，请联系开发者并提供详细的错误信息。

*   **下载失败**：
    *   请检查您的网络连接是否正常。
    *   确保您输入的 YouTube 视频链接是有效的。
    *   有时 YouTube 的下载策略会更新，可能导致下载功能暂时失效。请等待开发者更新版本。

感谢您的使用！
